# Curso Alura: JavaScript e Node.js - Criando a Primeira Biblioteca

## 📚 Sobre o Curso

Este repositório contém os materiais e projetos do curso **"JavaScript e Node.js: Criando a Primeira Biblioteca"** oferecido pela [Alura](https://cursos.alura.com.br/course/javascript-node-js-criando-primeira-biblioteca).

O curso foi desenvolvido para ensinar desde os conceitos fundamentais de JavaScript até a criação de bibliotecas práticas utilizando Node.js.

---

## 🎯 O que Fizemos no Curso

Durante o curso, desenvolvemos:

- ✅ Fundamentos de **JavaScript** e sua sintaxe
- ✅ Configuração e uso do **Node.js**
- ✅ Criação de **módulos e pacotes NPM**
- ✅ Desenvolvimento de uma **biblioteca personalizada**
- ✅ Publicação da biblioteca no **NPM** (npm registry)
- ✅ Implementação de **boas práticas** de desenvolvimento
- ✅ Versionamento e **controle de dependências**

---

## 📖 O que Aprendemos

### Conceitos Principais:

1. **JavaScript Avançado**
   - Closures e Escopo
   - Funções de Alta Ordem (Higher-Order Functions)
   - Manipulação de Objetos e Arrays
   - Promises e Async/Await

2. **Node.js**
   - Estrutura de projetos Node
   - Sistema de módulos CommonJS
   - Gerenciador de pacotes (npm/yarn)
   - Criação de scripts automáticos

3. **Desenvolvimento de Bibliotecas**
   - Organização de código reutilizável
   - Documentação de código
   - Testes unitários
   - Padrões de design

4. **Ferramentas de Desenvolvimento**
   - Git e GitHub
   - Versionamento semântico
   - Publicação em registros públicos

## 💻 Prática e Projetos

### Atividades Realizadas:

- 🔧 **Criação de módulos** com funcionalidades específicas
- 📝 **Escrita de testes** para validar o código
- 📦 **Empacotamento** do projeto como biblioteca
- 🌐 **Publicação** no NPM
- 📚 **Documentação completa** com exemplos de uso
- 🔄 **Atualizações e manutenção** da biblioteca

### Como Usar a Biblioteca:

```bash
# Instalar a biblioteca
npm install seu-pacote

# Importar em um projeto
const suaBiblioteca = require('seu-pacote');

# Usar as funcionalidades
suaBiblioteca.funcaoExemplo();
```

## 🚀 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Ericao327/Node-Js-Alura.git
   cd Node-Js-Alura
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Execute os testes:**
   ```bash
   npm test
   ```

4. **Use a biblioteca:**
   ```bash
   node seu-arquivo.js
   ```
